# CreditMitra Pro

CreditMitra Pro is a Next.js based loan management platform.

## Features

- Customer Registration
- OTP Login
- Customer Dashboard
- Loan Application
- Document Upload
- Admin Dashboard
- Loan Status Management
- Customer Profile Management

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Prisma ORM
- Supabase PostgreSQL
